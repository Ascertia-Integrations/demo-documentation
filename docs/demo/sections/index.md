Docusaurus can generate sidebar structure from the filesystem. A typical pattern is:

```text
docs/
  demo/
    sections/
      sections-and-subpages.mdx
      subpages/
        child-page.mdx
```

- Each folder becomes a **section**.
- Files inside folders become **pages**.
- Nested folders become **sub-pages**.
