title: CI + GitHub Pages
---

This repo deploys using the **platform reusable workflow**.

- On push to `main`: build + deploy current docs
- On push to `X.Y.Z`: sync version artifacts to `main`, then build + deploy

The workflow sets environment variables for Pages:

- `SITE_URL` (e.g. `https://<org>.github.io`)
- `BASE_URL` (e.g. `/<repo>/` for project pages)

`docusaurus.config.ts` reads these so deployments don’t require hardcoding.
